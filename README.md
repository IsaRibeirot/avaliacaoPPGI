# Avaliação PPGI

O Programa de Pós-Graduação em Informática (PPGI) da UFES precisa de um sistema de facilite o processo de avaliação de docentes credenciados ao programa. Tal cálculo é feito manualmente todo ano e a coordenação do programa gostaria de um software que automatizasse este cálculo.

O trabalho prático da disciplina de Programação III consiste em desenvolver o sistema computacional para solução do problema descrito na linguagem de programação apresentada durante a segunda metade do curso: Java.

---

O programa ainda conta com uma funcionalidade extra: uma interface Web que permite fazer o upload dos arquivos de entrada, gere os relatórios e os disponibilize para download. Essa funcionalidade foi construída com o framework [Spring](https://start.spring.io/), com estrutura retirada [deste guia](https://github.com/spring-guides/gs-uploading-files).

Para executar, basta ir até a aplicação principal contida na pasta equivalente e _run_.
No caso da aplicação Web, após a execução, ir até <http://localhost:8080/home>
